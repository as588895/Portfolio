# Aman Singh — Portfolio

This is a production-ready React + Vite portfolio template for Aman Singh.

Tech stack: React, Vite, Tailwind CSS, Framer Motion, React Icons, EmailJS

Getting started

1. Install dependencies

```bash
npm install
```

2. Run development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
npm run preview
```

EmailJS setup

- Create an account at https://www.emailjs.com/
- Create a service and an email template
- Replace `YOUR_SERVICE_ID`, `YOUR_TEMPLATE_ID`, and `YOUR_PUBLIC_KEY` in `src/components/Contact.jsx`

Deployment

Deploy the `dist/` folder to Netlify, Vercel, or any static host. For Vercel, connect the repository and set `npm run build` as the build command.

Notes

- Customize live and repo links in `src/components/Projects.jsx`.
- Tailwind config is in `tailwind.config.cjs`.
